# Linux Basics

## pwd
Prints the current working directory.

Example:
pwd

## ls
Lists files and directories.

Useful flags:
- ls -l
- ls -la

## cd
Changes directories.

Examples:
cd ~
cd ..
cd /home/andrew-thomas

## ls
### Purpose
List files and directories.

### Common Options

| Command | Description
|---------|---------------------------|
|ls       | Basic Listing             |
|ls -l    | Long Listing              | 
|ls -la   | Include hidden files      |
|ls -lh   | Human-readable file sizes |
